## How do I read the data?

### Contest List

The filename should be `list.txt`.

e.g.

```plain
Sleeping Cup #1
Sleeping Cup #2
Sleeping Cup #3
*
Sleeping Cup #4
Sleeping Cup #5
!
```

In this example, 5 contests are numbered:

| ID | Contest Name |
| -: | :- |
| 1 | Sleeping Cup #1 |
| 2 | Sleeping Cup #2 |
| 3 | Sleeping Cup #3 |
| 4 | **(No such contest)** |
| 5 | Sleeping Cup #4 |
| 6 | Sleeping Cup #5 |

### Rankings

The filename should be `rank.txt`.

e.g.

```plain
1 7 525
2 11 343
2 12 343
4 8 0
4 9 0
4 10 0
-1 -1 -1
```

In this example, 6 users are ranked:

| Rank | User ID | Rating |
| -: | -: | -: |
| 1 | 7 | 525 |
| 2 | 11 | 343 |
| 2 | 12 | 343 |
| 4 | 8 | 0 |
| 4 | 9 | 0 |
| 4 | 10 | 0 |
